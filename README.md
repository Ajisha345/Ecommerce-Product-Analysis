# Ecommerce Product Analysis Dashboard

## Overview
This is a Power BI dashboard created to analyze product data from an e-commerce dataset. It helps to understand product trends, pricing, and seller performance.

## Tools Used
Power BI, Excel

## Key Insights
- Most products belong to clothing category  
- Watches and wallets have higher prices  
- Some sellers have more product listings  
- There is a small relationship between price and rating  

## Features
- Category-wise product analysis  
- Price vs rating comparison  
- Top sellers overview  
- Interactive filters
